## ビジョン

fal'Cieは、日本語を含む複数言語に対応したオープンウェイト言語モデルです。最先端の自然言語処理技術を活用し、日本語ユーザーに高品質な言語処理能力を提供することを目指しています。

## ステータス

モデルウェイトは**近日公開予定**です。
